# LLRuby

LLRuby is LLVM-based JIT compiler for Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'llruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install llruby

## Usage

`require 'llruby'` is enough.
